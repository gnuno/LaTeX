# LaTeX para materias de la facultad
Repositorio oficial de archivos LaTeX para diferentes materias de la Universidad Nacional del Oeste.

## ¿Qué es LaTeX?
De acuerdo con la página del proyecto [The LaTex Project](https://www.latex-project.org/):
```LaTeX es un sistema de edición de texto de alta calidad; incluye caracteristicas diseñadas para la producción de documentación científica y técnica. LaTeX es el estandar para la comunicación y publicación de documentos científicos. LaTeX está disponible como software libre```

## ¿Para qué?
Éste es un repositorio pensado para compartir y administrar los cambios en documentos creados en el sistema de documentación LaTeX para las diferentes materias que se dictan en la Universidad Nacional del Oeste.
Algunas de nuestras ideas cuando creamos este repo fueron:
* **Aprender a usar LaTeX**: Siempre es lindo aprender algo nuevo, y sobre todo si te va a servir para el resto de la carrera y para el futuro como profesional. Ya sabemos que existen muchos editores de texto, pero LaTeX se usa mucho en el ámbito académico y científico y cuando le agarrás el gustito, después queres hacerlo todo en LaTeX. Acordate de lo que te digo.
* **Compartir apuntes**: Para nosotros es fundamental ésta idea. Queremos que se puedan compartir apuntes (notas de clases, resolución de ejercicios, guías de problemas, etc).
* **Resúmenes para parciales y finales**: Queremos que sea fácil poder guardar documentos que te sirvan posteriormente como resúmenes cuando te estés preparando para exámenes parciales o finales. Si bien lo ideal sería estudiar con **mucho tiempo de anticipación**, sabemos que muchas veces cuesta organizarse, quizas trabajás o simplemente pasas más tiempo del que quisieras en redes sociales.
* **Muchas cabezas, toman apuntes mejor que una sola**: Viste cuando llegas tarde a clases y estas en el fondo del aula y no escuchas mucho? o cuando los y las profes utilizan fibrones con tinta invisible y te cuesta ver lo que escriben, o simplemente te da vergüenza preguntar algún concepto que no pudiste anotar. No es genial poder tener apuntes de varios compañeros resumidos en un solo apunte? Los temas principales seguramente los copias, pero las meta-anotaciones que todos nosotros llevamos en nuestros apuntes pueden beneficiarnos en conjunto.
* **Aprender a usar github y git**: Si bien no es el objetivo principal, git y github son herramientas buenísimas para la colaboración y para la administración de cambios en archivos. La idea es que aprendas a usar estas herramientas, y ¿qué mejor manera de aprenderlas que utilizarlas? Pero no te preocupes si no las sabes usar, si necesitas un apunte, simplemente descargalo manualmente o copialo desde la página y guardalo en tu pc.

## ¿Cómo empezar?
Lo primero es instalar LaTeX en tu computadora. Existes muchos recursos en internet sobre cómo instalar LaTeX. Cuando lo hayas hecho, navegá a través de las carpetas y revisá si está el apunte que andas necesitando. Y por qué no, si te animás y tenés un apunte armado y querés compartirlo, adelante.

## Orden, arte y medida, para próspera vida
Obviamente para que este repo no sea un *CAOS*, necesitamos organizarnos:
* No malas palabras
* No apodos ofensivos
* Las sugerencias de edición se deben aprobar por nuestro staff para que se vean reflejadas.
* Nos reservamos el derecho de aprobar todos los cambios, pero de todas maneras, si hay algo que si o sí queres publicar, podes clonar este repo siempre respetando el mencionarnos.
* Los archivos que vamos a utilizar son **.tex** principalmente. Existen aplicaciones que te permiten escribir documentos LaTeX de manera sencilla y rápida que manejan sus propios archivos. En ese caso exportá tu archivo como tex si el programa te lo permite. En el futuro quizas agreguemos el archivo **.tex** y un archivo **.pdf**.
